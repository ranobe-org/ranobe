### Возможности

- Чтение новелл/ранобэ  
- Без рекламы и платных ограничений  
- Неограниченный обзор новелл  
- Многоязычные источники (вкл/выкл)  
  - Английский  
  - Русский  
- Поиск по ключевым словам  
- Поиск и сортировка глав  
- Вертикальное непрерывное чтение  
- Светлая и тёмная темы  
- Настройка чтения  
  - Изменение размера шрифта  
  - Изменение цвета фона и текста  
  - Бионическое чтение  
- Чтение без интернета и библиотека  
- История чтения

Разработчики этого приложения не имеют никакой причастности к контенту, доступному в приложении.
